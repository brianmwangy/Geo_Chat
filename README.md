# Geobots MVP
A local geospatial app with natural language querying using Ollama, FastAPI, and Next.js with Leaflet.

## Setup
1. Install prerequisites (Python, Node.js, Ollama).
2. Place GeoJSON datasets in `./data/`.
3. Follow backend and frontend setup steps below.

## Running
- Backend: `cd backend && uvicorn main:app --reload`
- Frontend: `cd frontend && npm run dev`
